# A secure JavaScript/TypeScript runtime built with V8, Rust, and Tokio

[deno](https://github.com/denoland/deno)  
[文档](https://deno.land/)

### install && demo
```bash
curl -fsSL https://deno.land/x/install/install.sh | sh
deno https://deno.land/welcome.ts
deno demo.ts

# 进入repl环境
deno
# 退出repl环境
exit
```
